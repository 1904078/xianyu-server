使用 spring boot + mybatis 搭建我的仿闲鱼app的 restful api 服务器

## 使用到的所有技术包括：

1. spring boot web框架
2. mysql 数据库
3. mybatis 持久层框架
4. logback 日志框架
5. gradle 构建工具
6. lombok 使用注解生成代码工具
7. idea 开发工具
8. restful风格的api
9. 使用postman测试api
10. 后期打算使用aop方式来处理日志
